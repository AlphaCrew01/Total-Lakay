# 🛠️ GUIDE D'ADMINISTRATION - TOTAL LAKAY

Ce guide est strictement réservé aux administrateurs de la plateforme. L'IA utilisera ces informations pour vous aider à gérer le site.

## 1. GESTION DES PRODUITS
- **Ajout rapide** : Utilisez le bouton "➕ Ajoute yon pwodui". Remplissez le nom, le prix (HTG), et l'URL de l'image.
- **Gestion du stock** : Le stock est décompté à chaque commande validée. Surveillez les alertes de stock bas.
- **Variations** : Vous pouvez ajouter des couleurs et tailles séparées par des virgules (ex: "Rouge, Bleu, Noir").

## 2. GESTION DES COMMANDES (LOGISTIQUE)
- **Changement de statut** : Cliquez sur "Jere Kòmand". Vous pouvez passer une commande de "An atant" à "Konfime", "Preparasyon", "Nan wout" ou "Livre".
- **Délais de livraison** : Vous pouvez définir un délai personnalisé pour chaque commande (ex: "24h", "2 jours").
- **Annulation** : Si vous annulez une commande, le stock est automatiquement réincrémenté.

## 3. GESTION DES CLIENTS & RÔLES
- **Promotion Admin** : Dans "Jere Kliyan", vous pouvez transformer un client en Administrateur. 
- **Attention** : Un administrateur a accès à toutes les données financières.
- **Diagnostic** : Si un client se plaint que l'IA ne répond pas, vérifiez la clé API dans "Configuration IA" et utilisez l'outil de diagnostic.

## 4. NOTIFICATIONS & COMMUNICATION
- **Mentions** : Vous pouvez envoyer des notifications ciblées :
  - `@all` : Tous les utilisateurs.
  - `@clients` : Uniquement les clients.
  - `@admins` : Uniquement l'équipe technique.
- **Barre de recherche** : Utilisez la barre de recherche dans le module de notification pour trouver un utilisateur spécifique.

## 5. ANALYSE FINANCIÈRE
- Le tableau de bord calcule le revenu brut basé sur les commandes livrées.
- Les graphiques montrent la progression des ventes par jour/mois.
