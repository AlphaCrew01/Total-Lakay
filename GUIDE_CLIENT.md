# 🛍️ GUIDE DU CLIENT - TOTAL LAKAY

Ce guide aide les clients à profiter pleinement de l'expérience Total Lakay. L'IA l'utilisera pour vous guider.

## 1. VOTRE COMPTE
- **Vérification** : Vous ne pouvez pas commander sans avoir cliqué sur le lien de vérification reçu par email.
- **Profil** : Enregistrez votre adresse et téléphone dans votre profil pour ne pas avoir à les retaper à chaque achat.

## 2. ACHATS & PAIEMENTS
- **MonCash** : C'est le moyen le plus rapide. Le paiement est instantané et sécurisé.
- **Cash** : Vous payez au livreur. Assurez-vous d'avoir le montant exact.
- **Panier** : Ajoutez plusieurs articles pour réduire les frais de livraison.

## 3. SUIVI DE COMMANDE
- **Statuts** :
  - **An atant** : On a reçu votre demande.
  - **Konfime** : C'est bon, on s'en occupe !
  - **Nan wout** : Ouvrez l'œil, le livreur arrive !
- **Notifications** : Activez les notifications sur votre téléphone (PWA) pour recevoir des alertes en temps réel.

## 4. FONCTIONNALITÉS AVANCÉES
- **Favoris** : Cliquez sur le ❤️ pour retrouver vos produits préférés plus tard.
- **IA Assistant** : Posez n'importe quelle question sur un produit (ex: "Est-ce que vous avez du riz en stock ?").
- **Avis** : Laissez une note (étoiles) et un commentaire après avoir reçu votre produit pour aider les autres clients.

## 5. INSTALLATION (PWA)
- **Sur Android** : Chrome -> Menu -> Installer l'application.
- **Sur iPhone** : Safari -> Partager -> Sur l'écran d'accueil.
- **Avantage** : Accès plus rapide, moins de consommation de données, et fonctionne hors-ligne.
